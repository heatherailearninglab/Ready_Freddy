Free AI Prompt Pack for Middle School Classrooms

20 Ready-to-Use Prompts for Grades 6-8

AI Kairos Learning Lab | aikairos.org

What this is

20 structured prompts designed for middle school classrooms. Each prompt includes a clear task, context, and suggested use. Print this pack, hand it out, and start using it today. No live AI tools required for discussion-based activities.

Teacher Quick-Start

Pick any prompt from this pack. Read it aloud or display it. Give students 2 minutes of think time. Then open discussion. You can also assign prompts as written warm-ups or exit tickets.

For prompts that involve writing to an AI tool: if students have access, let them try it. If they do not, have them write the prompt and predict what AI might produce.

Section 1: Understanding AI (Prompts 1-5)

Prompt 1
Explain what artificial intelligence is in three sentences. Use language a 5th grader could understand.
Use for: Warm-up, discussion starter
Prompt 2
List five places where AI already shows up in your daily life. For each one, explain what the AI is doing.
Use for: Partner talk, written response
Prompt 3
What is one thing AI can do well? What is one thing AI cannot do? Explain the difference.
Use for: Whole-class discussion
Prompt 4
A friend tells you, "AI knows everything." Do you agree or disagree? Write your response and explain your reasoning.
Use for: Written response, small group
Prompt 5
If AI does not think or understand, how does it produce answers that sound so smart?
Use for: Exit ticket, discussion

Section 2: Purpose and Responsibility (Prompts 6-10)

Prompt 6
Before using AI on a school assignment, what questions should you ask yourself? List at least three.
Use for: Warm-up, anchor chart creation
Prompt 7
A student uses AI to write their entire book report. Another student uses AI to help organize their notes before writing. What is the difference? Which is responsible? Why?
Use for: Small group debate, written response
Prompt 8
Describe a school task where using AI would help your learning. Then describe a task where using AI would get in the way of learning.
Use for: Partner talk, class discussion
Prompt 9
Write a short disclosure statement explaining how you used AI to help with a project. Include what tool you used, what you asked it to do, and what you did with the result.
Use for: Practice activity, exit ticket
Prompt 10
Should students always have to tell their teacher when they use AI? Why or why not?
Use for: Whole-class discussion, written opinion

Section 3: Writing Better Prompts (Prompts 11-15)

Prompt 11
Rewrite this weak prompt to make it stronger: "Tell me about the ocean."
Use for: Practice activity, partner work
Prompt 12
Using the FOCUS framework, write a prompt that asks AI to help you study for a science test on ecosystems. Include the function, objective, context, user, and specifics.
Use for: Individual practice, class modeling
Prompt 13
Two students ask AI the same question but one gets a better answer. What might the student with the better answer have done differently in their prompt?
Use for: Discussion, critical thinking
Prompt 14
Write a prompt asking AI to create a study guide for your upcoming social studies test. Be as specific as possible about what you need.
Use for: Individual practice
Prompt 15
Why does a vague question lead to a vague answer? What does that tell you about how AI works?
Use for: Exit ticket, reflection

Section 4: Checking and Evaluating (Prompts 16-20)

Prompt 16
AI writes a paragraph about the American Revolution. How would you check whether the facts are correct? List your steps.
Use for: Class modeling, written response
Prompt 17
What is an AI "hallucination"? Why is it a problem? Give an example of how it might affect a school project.
Use for: Discussion, vocabulary building
Prompt 18
You ask AI to describe a historical figure and the response only includes positive things. What might be missing? Why does that matter?
Use for: Bias discussion, critical reading
Prompt 19
Is it harder to spot a mistake when it is well-written? Explain why confident writing can be misleading.
Use for: Discussion, reflection
Prompt 20
Create a 3-step checklist you could use every time you get a response from AI. What would you check first, second, and third?
Use for: Individual activity, anchor chart

Student Prompt Checklist

Before you submit any AI-assisted work, check these:

☐ I had a clear purpose for using AI.
☐ I wrote a specific, structured prompt.
☐ I read the output carefully.
☐ I checked facts against a reliable source.
☐ I looked for missing perspectives or bias.
☐ I revised the output in my own words.
☐ I can explain every part of my final work.
☐ I am ready to disclose how I used AI.
